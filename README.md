# gulp-js

Базовые настройки для Gulp

## ШАГ 1

```
git clone git@github.com:vnepluev/gulp-js.git <папка вашего проекта>
```

<br>
## ШАГ 2
<br>
 Заходим внутрь папки с проектом и выполняем (на warning  внимание не обращаем)

```
npm install
```

<br>
## ШАГ 3
<br>
Всю разработку ведем в папке src
